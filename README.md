# VocalMind AI – Real-Time Voice Mentor

VocalMind AI is a real-time voice-first AI mentor powered by Murf Falcon Text-to-Speech technology.

Users can speak naturally with the AI system and receive instant voice responses.

## Features

* Real-time voice interaction
* Murf Falcon voice generation
* AI mentor responses
* Voice-based learning assistant
* Multilingual interaction

## Tech Stack

Frontend
HTML, CSS, JavaScript, Web Speech API

Backend
Node.js, Express

Voice AI
Murf Falcon Text-to-Speech API

## Architecture

User Voice → Speech-to-Text → AI Response → Murf Falcon TTS → Voice Output

## Installation

Clone the repository

git clone https://github.com/Jathin-stack/vocalmind-ai.git

Install dependencies

npm install

Create environment variables

MURF_API_KEY=your_api_key

Run the server

node backend/server.js
